   - display
   - float
   - clear
   - flex-box
   - z-index
   - position
   - overflow

 - Form:
   - `<input>` elements 
   - `type` attribute
   - `name` attribute
   - `<form>` element and it's attributes
   - `<label>`, `<textarea>` 
   - form validation
 - Colors and styling:
   - colors
   - opacity
   - gradients
   - box-shadow
   - outline
   - transition
 - Write clean CSS file 
 - Write semantic HTML file (SEO)
